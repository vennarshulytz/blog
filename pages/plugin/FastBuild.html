FastBuild：让你的编译快人一步，效率飙升！

今天，我想给大家介绍一款针对 Build的 idea 插件：FastBuild

作用

快速构建项目。 仅仅编译当前项目中所有“打开”并且发生“修改”的 Java 文件。

优势

  1. 和类似功能的插件相比，更加轻量、体积小，最新版仅仅 13.18 KB
  2. 和类似功能的插件相比，更加易于使用
  3. 和类似功能的插件相比，效率更高，不会造成 IDE 卡顿
  4. 兼容的 idea 版本更多（2017.3+）

安装

在File | Settings | Plugins中搜索 FastBuild

点击 Install 安装插件

注意：需要 idea 版本 大于等于 2017.3，才可在插件市场中搜索到。

用法

点击按钮即可
